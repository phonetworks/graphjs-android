### How to open project

 - Open Android Studio
 - Main menu: File -> New -> Import Project...
 - Go through simple wizard
 - Done

### How to run unit tests

 - Main menu: Run -> Edit configurations...
 - In opened dialog press top-left plus -> Android Instrumential Tests
 - Change name to `test`
 - Select module `graphjs`
